## 프로젝트 소개

<p align="justify">
  지금까지 개발하며 사용했던 Skill들을 종합적으로 적용해볼 수 있을거라고 생각하여 Community 프로젝트를 진행하게되었습니다. <br>
  일반적인 커뮤니티 사이트들과 비슷하게 여러 주제의 게시판이 존재하고, 게시글과 댓글로 소통하는 사이트입니다. <br>
  회원가입은 구글, 카카오, 네이버를 통한 소셜 로그인으로 가능합니다.
</p>

<br>
